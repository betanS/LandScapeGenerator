## Enunciado

Diseña un documento **HTML** que incluya código **JavaScript** que solicite el valor de cada variable —o la mayoría de ellas— extraídas de la *“imagen de la calle”* dada y genere una representación equivalente en el navegador.

El documento debe cumplir los siguientes requisitos:

- Usar **al menos cinco divisiones horizontales**, destinadas a:  
  1. Carteles (parte superior)  
  2. Número y puerta  
  3. Escaparates y ofertas  
  4. Reloj y semáforo (con color solicitado)  
  5. Coches

- Utilizar las funciones:  
  - `document.write()`  
  - `prompt()`  
  - `alert()`  
  - `console.log()`

- Usar imágenes de Internet para representar los elementos de la escena.
